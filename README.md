Binary Search Tree Visualization Using JavaFX

About
This project is a Binary Search Tree (BST) visualization tool built in Java using JavaFX. It allows users to insert, search and traverse nodes in a binary search tree and visually observe how the structure changes with each operation.

Features

Insert values into the BST interactively

Visual display of nodes and connecting edges

Search operation with visual highlighting

In-order, pre-order and post-order traversal

Real-time updates to the tree visualization

Clear and user-friendly interface

Technologies Used

Java

JavaFX for GUI

Maven for build and dependency management

JavaFX Scene Graph for rendering shapes and text

Getting Started

Prerequisites

JDK 11 or higher

Maven installed

JavaFX libraries, if not bundled

Installation

Clone the repository
git clone https://github.com/satishkumarpeddi/BinarySearchTree

Navigate into the directory
cd BinarySearchTree

Build the project
mvn clean install

Running the Application
Use the command:
mvn javafx:run

If the JAR file is generated, run it using:
java -jar target/BinarySearchTree-version.jar

Usage
After launching the application:

Enter any integer value in the input field to insert into the BST

The node will be placed in the correct position depending on BST rules

Use buttons to perform in-order, pre-order or post-order traversals

Enter a value in the search field to visually check if the value exists in the tree

Reset or clear the tree when needed
